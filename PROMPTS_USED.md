# Prompts and AI Tools Used

For this screening task, I primarily focused on building the core business logic, application architecture, and React components manually. This ensured the code quality met my standards and strictly aligned with the project requirements. However, to speed up repetitive development processes and focus on complex logic, I utilized **Google Gemini / DeepMind Assistant** as a pair-programming tool strictly for boilerplate generation and styling at scale.

### Instructions Given to AI Assistant:

1. **Types & API Setup:**
   *  " Task 1: Create lib/api.ts (getSnacks, getStudents, createStudent, etc...) Use arrays"
2. **React Query Provider Setup:**
   * "Task 2: Create app/providers.tsx. Initialize QueryClient, Wrap app. 
      Task 3: Wrap Layout in layout.tsx"

3. **Page Construction:**
   * "Task 4: Create /snacks page (Show name, price, orders count, 'Order' button). 
      Task 5: Create /students page (Show Name, referral code, total spent, 'View Details' button)"

4. **Final Enhancements:**
   * "Task 6: Persist Data (Orders, students to localStorage). 
   *  " Task 7: UI/UX (Improve UI, add Tailwind cards, grid layouts)." 
   * Task 8: Responsive Design.
   * "Fix: Navigation gap layout issues on mobile and desktop margins."
   * "Create README.md including Setup steps, Tech stack, Features, Folder structure."
